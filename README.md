# YouTube Clone Demo 📺

A high-fidelity recreation of the YouTube homepage interface, built entirely with **HTML5** and **CSS3**. This project demonstrates core front-end layout techniques without the use of JavaScript or external libraries.

---

## 🚀 Project Overview
The goal of this project was to replicate the complex UI components of YouTube, focusing on:
* **Grid Layouts:** Handling multiple video thumbnails across different screen sizes.
* **Navigation:** Crafting the sidebar and the search-focused header.
* **Component Design:** Styling video cards, user avatars, and action buttons.

## ✨ Key Features
* **Responsive Video Grid:** Uses CSS Grid to adjust columns dynamically.
* **Interactive Header:** Includes a functional-looking search bar and notification icons.
* **Sidebar Navigation:** A sticky sidebar for easy access to "Home," "Shorts," and "Subscriptions."
* **Category Filter Bar:** Scrollable tag buttons for content filtering.

## 🛠️ Built With
* **HTML5:** Semantic tags for better SEO and accessibility.
* **CSS3:** Custom Properties (Variables), Flexbox, and Grid for layout management.
* **Google Fonts:** Utilizing 'Roboto' to match the official YouTube aesthetic.

## 📁 File Structure
* `index.html` - The main entry point.
* `styles/` - Custom CSS files for layout and design.
* `image/` - Asset folder for thumbnails and profile pictures.
* `icon/` - SVG and PNG icons used throughout the site.
